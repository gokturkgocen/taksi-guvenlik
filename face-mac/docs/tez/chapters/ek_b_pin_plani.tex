\section{Güç ve Topraklama}

\begin{itemize}[itemsep=1pt]
    \item STM32 NUCLEO kartı USB üzerinden (ST-Link ya da USB User) beslenir.
    \item ESP32-CAM 5\,V Vin üzerinden beslenir; 3.3\,V regülatörü modülde
          dahilidir.
    \item HM-10 modülü 3.3\,V besleme alır (STM32'nin 3V3 pininden).
    \item Tüm topraklar ortak (GND) bara üzerinden bağlanır.
    \item ESP32-CAM tarafında flash sırasında IO0 GND'ye çekilir; sonrasında
          serbest bırakılır.
\end{itemize}
